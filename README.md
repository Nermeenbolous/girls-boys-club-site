# Girls & Boys Club Enrichment Site
## Site Description
[cite_start]This is a professional, multi-page website built for the Girls & Boys Club. [cite: 1, 13] 
It features semantic HTML5, responsive design for mobile and desktop, and meets 
[cite_start]accessibility standards to ensure all community members can access program information. [cite: 1, 4]

## List of Pages
* [cite_start]**Home**: An overview of our mission and community impact. [cite: 1, 13]
* [cite_start]**VolunTeen**: Details on community service opportunities for teens. [cite: 41, 42]
* [cite_start]**Adventures**: Information on overnight trips and exploration programs. [cite: 34, 35]
* **Contact**: A fully accessible inquiry form for parents and volunteers.
